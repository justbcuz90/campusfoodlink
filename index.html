Project Name

CampusFoodLink, but website is FoodFull.

Meet the Team

FoodFull is developed by Abdul & Allycia.

Experience

we have combined knowledge in reactjs, nextjs, and nodejs,python, html, css and vanilla javascript.

Menu items load from a JSON file and can be filtered by category. Login, cart, and checkout are handled in the browser for demo use.

This goal for this project, is to use HTML, CSS, JavaScript, JSON, and local browser storage data to create a food ordering experience.

Project features

Menu, account overview, vendor page, and checkout preview
