About FoodFull

Built for busy student life

This project uses HTML, CSS, JavaScript, and JSON data to create a practical front-end food ordering experience. Menu items load from a JSON file and can be filtered by category.

Login is handled with browser storage for demo use. Cart items persist in local storage, and checkout calculates totals instantly in the browser.

Project features

Menu, cart, login, checkout, review form

Designed to serve as a strong starter structure for a fuller CampusFoodLink build.
